# NaviQueue Pro

This is a Next.js application for a smart queueing system, built in Firebase Studio.

## Getting Started

To run the application in development mode, use the following command:

```bash
npm run dev
```

This will start the Next.js development server. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Installing as a Desktop App (PWA)

This application is a Progressive Web App (PWA), which means you can "install" it on your computer to run like a native desktop application. It will get its own icon on your desktop or taskbar and run in its own window.

### On Windows (using Chrome or Edge)

1.  Open the application's URL in your browser.
2.  Look for an "Install" icon in the address bar. It usually looks like a computer screen with a downward arrow.
3.  Click the icon and then click "Install" in the prompt that appears.
4.  The app will now be available from your Start Menu just like any other program.

## Deployment

This is a standard Next.js application and can be deployed to any platform that supports Next.js.

### Firebase App Hosting

This project is pre-configured for deployment with Firebase App Hosting. To deploy, you can connect your repository to a Firebase project.

### Other Platforms

You can also deploy this application to other services like Vercel, Netlify, or your own server.

1.  Build the application for production:
    ```bash
    npm run build
    ```
2.  Start the production server:
    ```bash
    npm run start
    ```
